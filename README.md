# MedGuide

Smart Analysis is an AI-powered healthcare platform designed to help users better understand their health through symptom-based analysis, medication management, condition tracking, and personalized healthcare insights. The platform provides a modern and intuitive experience for managing daily health routines and accessing important medical information.

## Overview

Smart Analysis enables users to:

- Analyze symptoms and receive AI-powered health insights
- Get personalized treatment recommendations
- Manage medications with smart reminders
- Track health metrics and progress
- Access information about common health conditions
- Maintain family health profiles
- Connect with healthcare providers
- Locate nearby pharmacies and medication resources



## Technology Stack

| Category | Technologies |
|-----------|-------------|
| Frontend | Next.js 15, React 19, TypeScript, Tailwind CSS 4 |
| UI Components | Radix UI, Lucide React, Sonner, Vaul |
| Forms & Validation | React Hook Form, Zod, Hookform Resolvers |
| Data Visualization | Recharts |
| Additional Libraries | Date-fns, Embla Carousel, Next Themes, Class Variance Authority |

## Installation

### Prerequisites

- Node.js 18 or later
- npm, yarn, pnpm, or bun


## Disclaimer

Smart Analysis is intended to provide educational and informational healthcare assistance. The platform does not replace professional medical advice, diagnosis, or treatment. Users should always consult qualified healthcare professionals regarding medical concerns and treatment decisions.
